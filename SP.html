Cyber Security

Cyber Security is the practice of protecting computer systems, networks, and data from theft, damage, unauthorizedaccess, and other cyber threats. It encompess a wide range of technologies, processes, and practices designed to safeguard digital information and ensure the confidentiality, integrity, and availability of data. In our project we use a machine learning algorithim to detect any netwrok intrusion with data provided from an outside source using this algorithim we can determine if a system has reliable network security.


Data Streams

A data stream refers to a sequence of data elements made available over time. In computing and information technology, data streams are commonly used to represent the flow of data that is continuously generated, processed, and consumed between the user and the server.


Malware

Malware which is short for malicious software, refers to any software intentionally designed to harm, exploit, or compromise the functionality of a computer, network, or device. Malware can take various forms, each with its own set of malicious objectives. There are many types of malware in our project we touch upon ransomware which is a type of malware that encrypts a user's files or entire system in some cases and causing them to lose access to said systems and demanding a ransom in return of controll of the systems or files.
